# Bricks-Breaker-in-C
벽돌깨기를 C 언어로 구현한 프로젝트입니다.

기본적인 컨셉은 엔드리스 벽돌깨기 게임입니다. 

간소하게나마 엔진을 구현하여 사용하였습니다 ( 유니티 엔진에서 사용되는 함수들을 참고하여 구현하였음. )
공의 움직임은 극좌표계(polar coordinate)를 활용하여 구현하였습니다.
